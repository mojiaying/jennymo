# vuepress 搭建快速搭建博客

# 环境准备

  1. node 安装 https://nodejs.org/en/download/
  2. 安装版本查看（有版本号表示node安装成功）： node -v
  3. npm 版本查看（node安装成功就会有npm版本）：npm -v

# 安装初始化

  1. 全局安装：npm i -g vuepress
  2. 创建项目文件目录：mkdir myblog
  3. 进入文件：cd myblog
  4. 项目初始化(生成一个package.json文件，这文件用于记录项目安装的依赖包和依赖包配置)：npm init -y
  5. 局部安装vuepress: npm i vuepress
  6. 在当前目录中创建docs目录

# 目录结构

  1. 在docs文件中创建 .vuepress 目录：cd docs > mkdir .vuepress
  2. 在.vuepress下创建总配置文件：cd .vuepress > touch config.js 内容示例：
module.exports = {
  title: '学习笔记',
  description: 'Just playing around',
  themeConfig: {
    logo: '/img/jenny.png',
    nav: [
      { text: 'Home', link: '/' },
      {
        text: '专题笔记', 
        items: [
          {text: 'unity', link: '/views/unity/index1/'},
          {text: 'C1', link: '/views/C1/'},
        ]
      }
      ],
    sidebar: {
      '/views/C1/': [
          {
            title: '基本语法',
            collapsable: true,
            children: [
              '/views/C1/index1', '/views/C1/index2',
            ]
          },
          {
            title: '高级教程',
            collapsable: true,
            children: [
               '/views/C1/index3', '/views/C1/index4'
            ]
          },
        ]
    }
  },
  markdown: {
    lineNumbers: true,
  },
  configureWebpack: {
    resolve: {
      alias: {
        '@alias': 'path/to/some/dir'
      }
    }
  }
}
