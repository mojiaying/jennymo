# 1.罗马数字转整数

罗马数字包含以下七种字符: I， V， X， L，C，D 和 M。

字符 数值
I 1
V 5
X 10
L 50
C 100
D 500
M 1000

例如， 罗马数字 2 写做 II ，即为两个并列的 1。12 写做 XII ，即为 X + II 。 27 写做  XXVII, 即为 XX + V + II 。

通常情况下，罗马数字中小的数字在大的数字的右边。但也存在特例，例如 4 不写做 IIII，而是 IV。数字 1 在数字 5 的左边，所表示的数等于大数 5 减小数 1 得到的数值 4 。同样地，数字 9 表示为 IX。这个特殊的规则只适用于以下六种情况：

I 可以放在 V (5) 和 X (10) 的左边，来表示 4 和 9。 X 可以放在 L (50) 和 C (100) 的左边，来表示 40 和 90。  C 可以放在 D (500) 和 M (1000) 的左边，来表示 400 和 900。 给定一个罗马数字，将其转换成整数。输入确保在 1 到 3999 的范围内。

解法一：

var romanToInt = function(s) {
    let list = {I:1, V:5, X:10, L:50, C:100, D:500, M:1000}
    let amount = 0
    for (let i = 0;i<s.length;i++){
        if(i === s.length -1) return  amount += list[s[i]]
        if(list[s[i]] < list[s[i+1]]){
            amount -= list[s[i]]
        } else {
            amount += list[s[i]]
        }
    }
}

# 最长公共前缀

编写一个函数来查找字符串数组中的最长公共前缀。

如果不存在公共前缀，返回空字符串 ""。

var longestCommonPrefix = function(strs) {
    if(strs.length === 0) return ''
    if(strs.length === 1) return strs[0]
    let flag = true
    let prefix = ''
    let i = 0
    while(flag){
        let p = strs[0][i]
        for(let j = 1;j<strs.length;j++) {
            if(!strs[j] || strs[j].length < i+1 || strs[j][i] !== p) {
                return prefix
            }
        }
        prefix += p
        i++
    }
};

# 3.二进制求和

给你两个二进制字符串，返回它们的和（用二进制表示）。

输入为 非空 字符串且只包含数字 1 和 0。

var addBinary = function(a, b) {
    let pre = 0
    a = a.split('').reverse()
    b = b.split('').reverse()
    let std = a.length > b.length ? a : b
    for(let i = 0;i < std.length;i++){
        let x = a[i] === undefined ? 0 : Number(a[i])
        let y = b[i] === undefined ? 0 : Number(b[i])
        let sum = x + y + pre
        if(sum < 2) {
            std[i] = sum
            pre = 0
        } else {
            std[i] = sum - 2
            pre = 1
        }
    }
    if(pre) {
        std.push(pre)
    }
    return std.reverse().join('')
};

# 4. 验证回文字符串

给定一个字符串，验证它是否是回文串，只考虑字母和数字字符，可以忽略字母的大小写。

说明：本题中，我们将空字符串定义为有效的回文串。

var isPalindrome = function(s) {
    s = s.match(/[0-9a-zA-Z]/g)
    if(s === null) return true
    let str = s.join('').toLowerCase()
    let len = str.length
    let n = Math.floor(str.length/2)
    for(let i=0;i<=n;i++){
        if(str[i] !== str[len - i - 1]) return false
    }
    return true
};

# 5.有效的字母异位词

给定两个字符串 s 和 t ，编写一个函数来判断 t 是否是 s 的字母异位词。

注意：若 s 和 t 中每个字符出现的次数都相同，则称 s 和 t 互为字母异位词。
方法一：

var isAnagram = function(s, t) {
    if(s.length !== t.length) return false
    let str
    while(s.length) {
        str = s[0]
        s = s.replaceAll(str, '')
        t = t.replaceAll(str, '')
        if(s.length !== t.length) return false
    }
    return true
};

# 9.仅反转字母

给定一个字符串 S，返回 “反转后的” 字符串，其中不是字母的字符都保留在原地，而所有字母的位置发生反转。

var reverseOnlyLetters = function(s) {
    s = s.split('')
    let lIdx = 0, rIdx = s.length -1, left = '', right = '', reg = /^[a-zA-Z]/
    function testA (idx) {
        if(reg.test(s[idx])) {
            return s[idx]
        } else {
            return ''
        }
    }
    while(lIdx < rIdx) {
        left = left || testA(lIdx)
        right = right || testA(rIdx)
        if(!left || !right) {
            if(!left) lIdx++
            if(!right) rIdx--
        }else if(right && left) {
            s[lIdx] = right
            s[rIdx] = left
            lIdx++
            rIdx--
            right = ''
            left = ''
        }
    }
    return s.join('')
};

返回 合并后的字符串 。

方法一：

var mergeAlternately = function(word1, word2) {
    let big = word1.length > word2.length ? word1 : word2
    let small = word1.length > word2.length ? word2 : word1
    let newStr = ''
    for(let i=0;i<small.length;i++){
        newStr += word1[i]
        newStr += word2[i]
    }
    newStr += big.slice(small.length)

};

# 11.长按键入

你的朋友正在使用键盘输入他的名字 name。偶尔，在键入字符 c 时，按键可能会被长按，而字符可能被输入 1 次或多次。

你将会检查键盘输入的字符 typed。如果它对应的可能是你的朋友的名字（其中一些字符可能被长按），那么就返回 True。

方法一：

var isLongPressedName = function(name, typed) {
    let ln = name.length
    let lt = typed.length
    if(ln>lt) return false
    let i = 0
    let j = 0
    while(i<ln&&j<lt) {
        if(name[i] === typed[j]) {
            i++
            j++
        } else if(typed[j] === typed[j-1]) {
            j++
        } else {
            return false
        }
    }
    while(j<lt) {
        if(typed[j] === typed[j-1]) {
            j++
        } else {
            return false
        }
    }
    return i === ln
};

# 12.增减字符串匹配

给定只含 "I"（增大）或 "D"（减小）的字符串 S ，令 N = S.length。

返回 [0, 1, ..., N] 的任意排列 A 使得对于所有 i = 0, ..., N-1，都有：

如果 S[i] == "I"，那么 A[i] < A[i+1] 如果 S[i] == "D"，那么 A[i] > A[i+1]

var diStringMatch = function(s) {
    let max = s.length
    let min = 0
    let res= []
    for(let i=0;i<s.length;i++) {
        if(s[i] === 'I') {
            res[i] = min
            min +=1
        } else {
            res[i] = max
            max -=1
        }
    }
    res[res.length] = max
    return res
};

var searchInsert = function(nums, target) {
    if(nums[nums.length-1] < target) return nums.length
    if(nums[0] > target) return 0
    for(let i=0;i<nums.length;i++) {
        if(nums[i] >= target) {

# 15.主要元素

数组中占比超过一半的元素称之为主要元素。给你一个 整数 数组，找出其中的主要元素。若没有，返回 -1 。请设计时间复杂度为 O(N) 、空间复杂度为 O(1) 的解决方案。

var majorityElement = function(nums) {
    if(nums.length === 1) return nums[0]
    let obj = {}
    let target = Math.floor(nums.length/2)
    for(let i=0;i<nums.length;i++) {
        if(obj[nums[i]]) {
            if(obj[nums[i]] === target) return nums[i]
            obj[nums[i]] += 1
        } else {
            obj[nums[i]] = 1
        }
    }
    return -1
};

        obj[nums[i]] = 1
    }
